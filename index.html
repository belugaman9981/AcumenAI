Get started in 30 seconds

Clone, configure, run.

# Clone the repo
git clone https://github.com/yourusername/AcumenAI.git
cd AcumenAI/local-ai-agent

# Install dependencies
pip install -r requirements.txt

# Set your API key in config.py, then run
python main.py

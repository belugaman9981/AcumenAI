Quick start

Up and running in 60 seconds

Clone, install, run. No configuration needed.

# Clone the repo
git clone https://github.com/belugaman9981/AcumenAI.git
# Go to the folder with api_server.py
cd your/path/to/local-ai-agent

# Install and run (or set the path in Settings)
pip install -r requirements.txt
python api_server.py
